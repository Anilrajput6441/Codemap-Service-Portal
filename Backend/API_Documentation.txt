GET request:

/api/v1/technologies/ - Get List of all technologies
/api/v1/services/ - Get a list of all services  - Done
/api/v1/projects/ - Get a list of all projects
/api/v1/teams/ - Get a list teams
/api/v1/queries/ - Get a list of all queries - Done
/api/v1/qoutes/ - Get a list of all  qoutes


POST request:
/api/v1/technologies/ -Post a specific technology
/api/v1/services/ -Post a specific service -Done
/api/v1/queries/ -Post a specific query -Done
/api/v1/qoutes/ -Post a specific qoutes
/api/v1/projects/ -Post a specific project
/api/v1/teams/ -  Post a specific team member
